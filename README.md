# DataPilot

An interactive machine learning workbench built with Streamlit that helps users explore datasets, analyze data quality, visualize statistics, and perform machine learning tasks.

## Current Status
🚧 Phase 1 Completed – Data Explorer

### Features
- Upload CSV and Excel files
- Dataset preview
- Column explorer
- Basic statistics
- Data quality report

## Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Scikit-learn

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
